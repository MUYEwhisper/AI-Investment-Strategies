智能体合集：

1️⃣ 投资问答助手

名称：智投问答（SmartQ Invest）
简介：
面向投资场景的智能问答助手，支持用户随时提出投资相关问题，从市场判断、标的选择到风险分析，提供结构化、专业化的即时解答，作为整个系统的统一入口。

2️⃣ 风险评估智能体

名称：风控雷达（Risk Radar AI）
简介：
通过多维数据分析对投资标的进行风险扫描，评估波动性、回撤概率及潜在不确定因素，输出直观的风险等级与预警提示，为用户构建安全边界。

3️⃣ 投资方向筛选智能体

名称：方向导航（Sector Navigator）
简介：
基于宏观经济环境与行业发展趋势，筛选具备增长潜力的投资方向，帮助用户在众多赛道中快速锁定高价值领域。

4️⃣ 市场趋势判断智能体

名称：趋势洞察（Market Insight AI）
简介：
对整体市场环境进行动态分析，识别当前市场处于上行、震荡或下行阶段，为投资决策提供宏观层面的时机判断依据。

5️⃣ 买卖时机判断智能体

名称：时机捕手（Timing Hunter）
简介：
结合技术指标与市场行为，识别关键买卖节点，给出合理的入场、止盈与止损建议，帮助用户把握交易节奏。

6️⃣ 投资组合优化智能体

名称：资产配置引擎（Portfolio Engine）
简介：
根据用户风险偏好与资金规模，自动生成科学的资产配置方案，实现收益与风险的动态平衡，提升整体投资效率。

7️⃣ 持仓分析与调仓建议智能体

名称：持仓管家（Position Manager AI）
简介：
实时分析用户当前持仓结构，评估资产健康度，并提供调仓、加仓或减仓建议，实现持续优化管理。

8️⃣ 投资决策解释智能体

名称：决策解读官（Explainable AI Advisor）
简介：
将复杂的模型结果转化为清晰易懂的语言解释，帮助用户理解每一项投资建议背后的逻辑与依据，提升系统透明度与信任感。

9️⃣ 情景模拟智能体

名称：投资沙盘（Scenario Simulator）
简介：
构建多种市场情景（如上涨、下跌、突发事件），模拟不同条件下的收益与风险变化，帮助用户提前预判可能结果。

🔟 投资策略生成智能体

名称：策略工坊（Strategy Lab AI）
简介：
根据用户目标与偏好，自动生成个性化投资策略，包括长期配置、定投方案及短期交易策略，提供可执行的行动路径。

二、整体项目简介（可直接用于申报书/答辩）

项目名称（可选）：
智能投资决策多智能体系统（Multi-Agent Investment Decision System）

项目简介：

本项目面向复杂多变的投资决策场景，构建了一套由多个智能体协同驱动的智能投资决策系统。系统围绕“市场分析—标的筛选—风险评估—策略制定—持仓优化”的完整决策链，将投资过程拆解为多个专业化子任务，并由不同功能的AI智能体分别承担。

在系统中，趋势洞察者负责宏观市场判断，方向导航仪筛选潜在投资领域，风控雷达对风险进行精准评估，时机捕手辅助交易决策，资产配置引擎优化整体组合结构，持仓管家持续跟踪资产状态。同时，投资沙盘提供情景模拟能力，策略工坊生成个性化投资方案，最终由决策解读官对结果进行可解释化输出，提升用户理解与信任。

通过多智能体之间的协同工作，系统能够从多个维度对投资问题进行综合分析，输出结构化、可执行且具备可解释性的决策建议。相比传统单一模型，本项目在决策完整性、风险控制能力以及用户交互体验方面具有显著优势。
